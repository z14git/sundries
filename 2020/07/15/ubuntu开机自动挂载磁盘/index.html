ubuntu开机自动挂载磁盘

发表于 2020-07-15 更新于 2020-09-07

格式化磁盘

用Gparted

创建挂载目录

mkdir ~/data

设置开机自动挂载

查询硬盘分区对应的UUID

eg:

修改文件

sudo vim /etc/fstab

增加一行：

第一个数字：0表示开机不检查磁盘，1表示开机检查磁盘；
第二个数字：0表示交换分区，1代表启动分区（Linux），2表示普通分区

最后，执行sudo mount -a自动挂载

权限问题

这种方法挂载后，普通用户是不能写的，暂时的解决办法是执行 sudo chmod 777 ~/data
