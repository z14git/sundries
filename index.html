发表于 2022-11-03

创建并运行一个新的docker容器

命令解析：

  1. run 新建一个窗口并且运行
  2. -i 交互式，连接网络
  3. -t 打开终端
  4. –gpus all 使用所有显卡
  5. 8888:8888 将容器的8888端口映射到wsl的8888端口
  6. –name torch 将该容器命名为 torch
  7. -v 将本地 /home/z14/repos 目录映射到容器的 /workspace/repos
  8. –ipc=host 使用主机内存
  9. pytorch/pytorch:1.13.0-cuda11.6-cudnn8-runtime 容器使用的镜像

How To Disable Unattended Upgrades On Ubuntu

发表于 2021-11-25

有时在使用 apt install xxx 时会出现以下错误信息：

而且在关机时还会出现：

这就是 unattended-upgrades 服务造成的。

解决方法

运行：

sudo dpkg-reconfigure unattended-upgrades

选择 No。

又或者直接移除：

sudo apt remove unattended-upgrades
